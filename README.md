## Installation

You can install the library via pip:

```shell
pip install huggingface-datasets-cocoapi-tools
```

You can also install the library with the optional dependencies for [ðŸ¤— huggingface datasets](https://github.com/huggingface/datasets):

```
pip install 'huggingface-datasets-cocoapi-tools[datasets]'
```
